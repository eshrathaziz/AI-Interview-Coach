# 🚀 Overview

AI Interview Coach is a full-stack web application that leverages **Google Gemini AI** to simulate technical interviews, evaluate resumes, calculate ATS scores, match resumes with job descriptions, and generate personalized interview feedback.

The platform enables candidates to prepare for real-world interviews through AI-powered conversations, coding challenges, resume optimization, and performance analytics.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/eshrathaziz/AI-Interview-Coach.git
```

## Install Dependencies

```bash
npm install
```

## Configure Environment

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key
```

## Run the Application

```bash
npm run dev
```
